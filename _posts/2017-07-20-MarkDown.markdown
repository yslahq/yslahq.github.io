# 列表

## 普通无序列表

*   列表文本前使用 [减号+空格]

*   列表文本前使用 [加号+空格]

*   列表文本前使用 [星号+空格]

## 普通有序列表

1.  列表前使用 [数字+空格]
2.  我们会自动帮你添加数字
3.  不用担心数字不对，显示的时候我们会自动把这行的 7 纠正为 3
